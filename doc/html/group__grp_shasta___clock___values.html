Detailed Description

The CPU subsystem has multiple dividers for the clocks of different modules and some of the clocks shall run at a fixed frequency. The according values are summarized here.

#define FREQ_OTP_CLOCK   ((uint32_t) 25)

#define SYS_ALPHA_CLK_DIVIDER   (uint8_t) (FREQ_PRIMARY_CLOCK / FREQ_ALPHA_CLOCK)
