Lauterbach Debugger

Introduction

For debugging firmware the Shasta project is using the Lauterbach Debugger.
There is plenty of documentation in the debugger manuals itself. What is covered here is how to generate hardware register descriptions in per files.

Per File Generation

The per file generation is performed in LINUX with metagen.

  • Steps in LINUX
  1. goto /var/vob/shasta/shasta/vob/bin
    make hals # needed to get SYSTEM.xml
    make pers
  2. Commit and push changes in the git repository (affected folders):
    • shasta_fw/tools/scripts/per_files/per_filtered
    • shasta_fw
  • Steps in WINDOWS:
  1. pull git repo
  2. execute shasta_fw/projects/t32_rom_debug/t32_start.bat this will start Lauterbach
  3. Check if the per file can be loaded in Lauterbach with per.program ...\tools\T32\per\per_infineon_shasta.per
  4. if there is an error modify convert_per.py accordingly
