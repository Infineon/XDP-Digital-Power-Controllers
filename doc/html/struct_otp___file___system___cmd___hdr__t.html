Detailed Description

OTP file system command header structure

Field Documentation

uint16_t cmd

Lower 8 bits File system command
Higher 8 bits: Xvalent selector (supported field selection: Otp_File_System_Cmd_Types_t).

uint32_t crc_hdr

CRC-32 value of the command header blocks (e.g. Otp_File_System_Cmd_Hdr_t + Patch_Cmd_Image_t). This checksum field is ignored in case it is set to zero. The checksum calculation includes the Otp_File_System_Cmd_Hdr_t header fields 'cmd', 'option', 'size' and 'size_red'. It also includes all field of any sub-command header (e.g. Patch_Cmd_Image_t for command OTP_FS_CMD_PATCH_IMAGE).

Files with a wrong checksum value are ignored and the following file system commands are processed.

uint16_t option

Option field. Supported bit values are:

  • OTP_FS_PATCH_ABSOLUTE
uint16_t size

Overall file system command size in Bytes (includes the command and size parameter).

uint16_t size_red

Overall file system command size in Bytes (includes the command and size parameter).
