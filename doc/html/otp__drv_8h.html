)

This function reads the dedicated OTP instance. It allows accessing the OTP cross boundary for any Byte count granularity.

The hardware restriction where only whole OTP rows are read or written is encapsulated and hidden by this function.

Parameters
sectionPointer to corresponding OTP section
offsetByte offset inside the OTP where to read or write the data
ptrRAM data pointer where to read the written data, or to write the read data
numNumber of Byte to read or write
Exception:
TIMEOUT.
