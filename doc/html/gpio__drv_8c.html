void GPIO_Drv_init ( gpio_idx_t  Bank)

This function initializes the GPIO instance/driver. All members of the GPIO instance bank are initialized. All interrupt pending for the GPIO instance bank are reset.

Parameters
Bankpoints to the GPIO bank to be initialized.
Returns
None

Setup GPIO's pins for the active bank specified by Bank

void GPIO_Drv_uninit ( gpio_idx_t  Bank)

This function uninitializes the GPIO instance/driver. All members of the GPIO instance bank are reset.

Bankpoints to the GPIO bank to be initialized.
Returns
None

void GpioBank_RstCnfg ( gpio_idx_t  Bank)

Resets the configuration (used pin) of the specified GPIO bank.

Parameters
Bankis the bank number of the GPIO to operate on. Valid values are 0 to GPIO_MAX_BANKS-1.
Cnfgspecifies the gpio bit being release with bit 0 being lsb.
Note
This function is used to release a gpio bit. The bit is removed from all FW configured pins of Gpio_drv_app.GpioCnfgDone.
