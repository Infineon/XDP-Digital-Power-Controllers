Detailed Description

OTP file system command handler mapping. A callback handler is mapped to handler OTP file system command of the given command type.

Field Documentation

Otp_File_System_Cmd_Types_t cmd_type

File system command type where to call the handler callback to handle such commands.

File system command handler to call in case a valid cmd_type file system command is found in OTP.
