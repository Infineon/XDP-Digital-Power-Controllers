Parameters
looppmbus page to update status on
fault_typeFault type to perform lookup from. Has a _FAULT_DESC_s structure associated with this fault_type.
fault_response_cmd_bytefault response byte //Response[7:6], Retry_Setting{5:3] and Delay_Time[2:0]
Config_Delay_UnitThis is used to decode the fields in FW_CONFIG_FAULTS PMBUS command (Time unit for retry responses)
response_configThis is used to decode response command byte. Voltage(Temp, TON MAX) type or CURRENT type response
pmbus_status_commandstatus command number STATUS_XXX used for reporting the fault on pmbus
pmbus_status_bit_positionbit position in command STATUS_XXX used for reporting the fault on pmbus
hysteresissets the hysteresis FAULT_TYPE_t(=0 - no hysteresis[no wait till condition clears - retry after Delay time], =fault_type - sets hysteresis[waits till condition clears then retry after Delay time])
