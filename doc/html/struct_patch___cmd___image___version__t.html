Detailed Description

This structure defines the command parameter for the OTP_FS_PATCH_IMAGE_VERSION command. The Patch_Cmd_Image_Version_t structure data are followed by the patch data. The amount of data is calculated as Patch_Cmd_t.size - sizeof(Otp_File_System_Cmd_Hdr_t) - sizeof(Patch_Cmd_Image_Version_t).

Field Documentation

uint32_t execute

Target OTP entry address where to execute the patch image (callback function).

uint32_t location

Target OTP address where to store the overall patch (including the patch command header).

uint32_t version_base

Firmware version UTC timestamp where the patch can be applied on. The patch is not applied and discarded in case the current firmware has a different UTC timestamp.

uint32_t version_new

New firmware version information. The current firmware UTC timestamp information is updated based on the patch information.
