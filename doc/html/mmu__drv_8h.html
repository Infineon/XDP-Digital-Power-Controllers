Version History:
Version 0.03: Restructuring of the BIF per register protection The BIF_PER register protection can now be enabled per sub-block
Version 0.04: RAM reduced to 2x16kbyte new shasta_hal_mmu.h Version 0.05: ROM increased to 80kbyte Test, Teststat, cse registers added new shasta_hal_mmu.h
