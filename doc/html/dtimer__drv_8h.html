Version definitions for dtimer

  • Version 0.05: Timer did not work correctly with a load of 0 In this case an interrupt is issued before the initialization was finished.
  • Version 0.06: Fast Timer start added, see JIRA Shasta-619

enum dtimer_instance_e

Three dtimer modules are available with two timers each. The first module contains DTIMER0 and DTIMER1. The second module contains DTIMER2 and DTIMER3. The third module contains DTIMER4 and DTIMER5.
