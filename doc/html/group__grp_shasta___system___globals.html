Detailed Description

System global boot resistant variables, which are cross module shared all over the system. They are not initialized during system startup.

Variable Documentation

uint32_t sys_boot_descriptor

Used in case of errors in the HW modules of the CPU subsystem. It contains information about which modules (HW + corresponding FW) of the CPU subsystem shall be disabled. Furthermore, it contains information about how clocking and timing shall be set. See sys_boot_descr_format.h for details.

Osys_Boot_State_t sys_boot_state

State of the boot sequence.

uint8_t sys_num_critical_reboots

Counter which is increased for each reboot that was caused by a CRITICAL error.

uint32_t SystemCoreClock

System Clock defines the clk of the CPU in Hz.
