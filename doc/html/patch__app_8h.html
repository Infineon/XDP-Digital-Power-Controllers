void patch_otp_process ( Otp_Partition_Selector_t  partition)

The function patch_otp_process scans the OTP partition for an existing patch and executes it. All patches which are set to invalid are ignored.

This function could either process a full image patch (after loading the CPU is reset) or a partial patch which is loaded and executed without reset.
Depending on the the Otp_File_System_Cmd_Hdr_t::option settings either multiple patches or only a single patch can be loaded from a partition.
