void Gpio_IntrHandler ( void  )

This function is the interrupt handler for GPIO interrupts. It checks the interrupt status registers of all the banks to determine the actual bank in which an interrupt has been triggered. It then calls the upper layer callback handler set by the function GpioBank_SetCallbackHandler(). The callback is called when an interrupt sets the interrupt status register.

Returns
None.
Note
This function does not save and restore the processor context such that the user must provide this processing.

void GpioBank_IntrClear ( gpio_idx_t  Bank,
uint32_t  Mask 
)

This function clears pending interrupt(s) with the provided mask. This function should be called after the software has serviced the interrupts that are pending.

Parameters
Bankis the bank number of the GPIO to operate on. Valid values are 0 to GPIO_MAX_BANKS-1.
Maskis the mask of the interrupts to be cleared. Bit positions of 1 will be cleared. Bit positions of 0 will not change the previous interrupt status.
Note
None.

void GpioBit_IntrType ( gpio_idx_t  Bank,
uint8_t  Cnfg,
uint8_t  Type 
)

This function sets the Type for the specified interrupt pin of the specified bank.

Parameters
Bankis the bank number of the GPIO to operate on. Valid values are 0 to GPIO_MAX_BANKS-1.
Cnfgis the bit of the pin for which the Type is to be set. Bit position of 1 will be specified. Bit positions of 0 will keep the previous setting.
Typesare 0, 1, 2, 3, and 4 where: 0 = GPIO_IRQ_TYPE_EDGE_BOTH 1 = GPIO_IRQ_TYPE_EDGE_RISING 2 = GPIO_IRQ_TYPE_EDGE_FALLING 3 = GPIO_IRQ_TYPE_LEVEL_HIGH 4 = GPIO_IRQ_TYPE_LEVEL_LOW
