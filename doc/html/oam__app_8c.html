void Default_IRQHandler ( void  )

The default handler for all those IRQs that do not have a dedicated handler yet but where still enabled by mistake. It acknowledges and disables the IRQ.

void HardFault_Handler ( const Cortex_M0_Stack_Frame_t *  stack_frame)

Does cleanup and diagnostic work in the catastrophic event of a Hard Fault on the Cortex-M0 processor. Specifically it creates an exception log with the program counter of the instruction which should be executed after the one causing the Hard Fault and resets the system. See the List of Errors for all errors causing a Hard Fault.

Parameters
stack_frameStack frame that was stored when executing the instruction causing the Hard Fault. This parameter already contains the correct stack which is dependent on the current execution level: "main stack" for the boot sequence and OS kernel; "process stack" for thread-level (see HardFault_PreHandler() in startup_shasta.)
