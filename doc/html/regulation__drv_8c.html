void iout_temperature_compensation ( PMBUS_PAGE_t  loop)

IOUT APC temperature compensation. Use this to modify current measurements according to temperature. This will allow the iout isen gain to be modified based on changes in temperature Register this with regulation state machine like so: Regulation_set_regulation_event_cb(REGULATION_STATE_AT_TARGET_VID, REGULATION_CONTROLS_TELEMETRY_UPDATED, iout_temperature_compensation); Using REGULATION_CONTROLS_TELEMETRY_UPDATED will call this function every 1ms (sample rate of telemetry thread).

void Regulation_Get_All_Q_Messages_By_Priority ( void  )

This is used to grab all rtos messages for regulation sm and push to local queue. Then it reorders the queue for the highest priority message first. This only reorders 1 message to the front so it needs to be called in the regulation sm while loop. Works in conjunction with regulation sm while loop and Regulation_Get_All_Q_Messages_By_Priority function
